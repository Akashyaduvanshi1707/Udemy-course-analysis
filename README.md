# Udemy Course Analysis - Python EDA Project

## Project Overview
Exploratory Data Analysis (EDA) project performed on Udemy course datasets using Python libraries such as Pandas, Matplotlib, and Seaborn.

The project focuses on analyzing course categories, pricing, subscribers, reviews, and publishing trends to uncover meaningful business insights.

---

## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Key Analysis Performed
- Course category analysis
- Paid vs free course comparison
- Subscriber trend analysis
- Course pricing analysis
- Popular subject identification
- Published year trend analysis

---

## Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Visual data representation
- Trend and pattern identification
- Business insights generation

---

## Dataset
Udemy courses dataset containing:
- Course titles
- Subscribers
- Reviews
- Pricing
- Subject categories
- Published timestamps
